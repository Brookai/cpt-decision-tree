# Brook CPT Coding Decision Tree

An interactive, informational reference that renders the CPT coding decision tree for
Remote Patient Monitoring (RPM) and care-management (CCM / PCM / APCM) services.

**Live page:** https://brookai.github.io/cpt-decision-tree/

## Scope

- Informational / decision-support only. This page does **not** bill and does **not**
  optimize allocation.
- Rates shown are **representative CPT reimbursement rates** on a CY2026 Medicare
  Physician Fee Schedule basis. They are **not** partner-contracted rates.
- Contains **no PHI** and **no partner-specific or proprietary financial data**.

## Contents

- `index.html` — the self-contained artifact (all CSS/JS/data inlined; no external
  dependencies). Open it directly in any browser, or view the hosted page above.
